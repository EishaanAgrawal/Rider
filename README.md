# Uber Clone

A full-stack ride-hailing platform inspired by Uber that allows users to book rides, track drivers in real time, estimate fares, and manage trip history.

## Features

### User Features

* User Registration & Login
* Location Search
* Ride Booking
* Fare Estimation
* Real-Time Driver Tracking
* Trip History

### Driver Features

* Driver Registration
* Accept/Reject Ride Requests
* Ride Status Updates
* Earnings Dashboard

### Admin Features

* Manage Users
* Manage Drivers
* Ride Analytics

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Redux Toolkit

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Real-Time Communication

* Socket.IO

### Maps & Location

* Google Maps API

### Authentication

* JWT
* bcrypt

## System Highlights

* Real-time ride matching
* WebSocket communication
* Secure JWT authentication
* Scalable REST APIs
* Location-based services
